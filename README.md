# Munirah Blog App

This is a simple blog application built with **React** and **TanStack Router**.

## Features

- Home page with a list of blog posts
- Dynamic Post Detail pages
- Back navigation
- 404 fallback page
- Lazy loading with React Suspense
- Error boundary for reliability
- Clean and responsive styling with CSS

## Technical Details

- **Routing:** TanStack Router (type-safe, modern alternative to React Router)  
- **Styling:** Custom CSS  
- **React Features:** Suspense, lazy loading, functional components

## How to Run

1. Clone the project or open in StackBlitz
2. Install dependencies: `npm install`
3. Start the app: `npm run dev` (or `npm start` depending on IDE)

## Notes

This project demonstrates understanding of routing, dynamic pages, error handling, and clean front-end design.
